# Oaxaca Website Project

## Overview
This project is a comprehensive website dedicated to showcasing the rich cultural heritage, delicious cuisine, vibrant tourism, and exquisite handicrafts of Oaxaca, Mexico. The website is built using HTML, CSS, and SASS, and incorporates animations to enhance user experience on every page.

## Features
- **Responsive Design:** The website is fully responsive, ensuring it looks great on all devices, from desktops to mobile phones.
- **Animations:** Smooth and engaging animations are used throughout the site to create a dynamic user experience.
- **Detailed Sections:** The site is divided into clear sections covering the state's history, geography, population, food, tourism, and handicrafts.
- **High-Quality Images:** Stunning visuals are used to capture the beauty and essence of Oaxaca.

## Technologies Used
- **HTML:** The structure of the website is created using semantic HTML5 elements for better accessibility and SEO.
- **CSS:** Custom styles are applied using CSS to ensure a visually appealing design.
- **SASS:** The project uses SASS for more efficient and organized styling, allowing for reusable variables, mixins, and nested rules.
